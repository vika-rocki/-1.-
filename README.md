## Коды для Git:

1. **pwd** - показать путь (где я сейчас)
2. **~** -  обозначение домашней папки. Попасть в нее ```cd ~
```
3. **cd** - переход в другую папку 

Нужно ввести название папки, или ~ для перехода в домашнюю папку. 

Переместиться через несколько папок, то есть внутрь конечной ```cd github/open-source-project
```

Вернуться в родительскую папку — то есть на уровень выше ```cd ..
```

Если в названии папки есть пробелы, нужно использовать кавычки. ```cd "Фотографии с дня рождения"
```
